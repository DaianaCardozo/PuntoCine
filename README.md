# Punto Cine

Punto Cine es una aplicación web inspirada en la experiencia digital de un cine. El proyecto presenta una cartelera de películas, secciones promocionales, información destacada y un formulario de contacto, con una estructura clara, diseño responsive y funcionalidades desarrolladas con JavaScript.

## Descripción

El objetivo del proyecto fue construir una interfaz moderna, funcional y adaptable a distintos dispositivos, aplicando buenas prácticas de HTML semántico, CSS y JavaScript.

La página está organizada con una estructura semántica clara mediante etiquetas como `header`, `nav`, `main`, `section` y `footer`, favoreciendo la accesibilidad, el orden del contenido y el mantenimiento del código.

## Funcionalidades

- Cartelera de películas.
- Secciones promocionales.
- Formulario de contacto.
- Diseño responsive.
- Navegación clara entre secciones.
- Interacciones desarrolladas con JavaScript.
- Estructura semántica en HTML.
- Estilos personalizados con CSS.

## Tecnologías utilizadas

- CSS3
- JavaScript
- Git
- GitHub
